# Sorting-simulator
A simulator to visualize sorting algorithms using Java.
This project is based on Java's AWT, Swing and Multithreading concepts.

# Supported sorting algorithms
1. Selection Sort
2. Bubble Sort
3. Cocktail sort
4. Odd Even sort
5. Insertion Sort
6. Tim sort
7. Quick sort
8. Heap sort
9. Merge sort
10. Pigeonhole sort
11. Radix sort(MSB)
12. Bogo sort
